# 乔木电影清单 Design System：夜场影厅

## 1. 视觉主题与氛围

「夜场影厅」把网站读成一个中文电影片单工具，而不是营销页。页面必须保持合适网页宽度，不做全屏首屏，不做大面积空洞 Hero。暗场是为了让海报、片名、评分和观影入口成为视觉焦点。

气质关键词：午夜场、电影节、放映单、海报墙、低照度、清晰可读。

功能契约：用户 3 秒内必须知道这里是 IMDb Top 250 电影清单；必须能搜索、扫列表、进入详情；必须能看到收藏、看过、想看；公开 UI 不出现接口、模型、技术栈名。

## 2. 色板与角色

- `--ink: #f7f3ea`：暗底主文字，不用纯白刺眼。
- `--muted: rgba(247, 243, 234, .68)`：次级文字，暗底不得低于 .60。
- `--faint: rgba(247, 243, 234, .46)`：极次级标签，只用于短 meta。
- `--background: #101419`：页面暗场底色，不用纯黑。
- `--surface: #171b21`：主卡片和详情面板。
- `--surface-2: #1f242b`：交互控件、输入框、状态按钮。
- `--line: rgba(247, 243, 234, .14)`：暗场细边框。
- `--accent: #e0b86a`：放映机暖黄，用于搜索按钮、焦点、重要徽标。
- `--accent-ink: #17110a`：暖黄按钮上的文字。
- `--danger: #e06a6a`：错误和取消语义。

## 3. 排版规则

中文正文和 UI 使用系统中文栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Noto Sans SC", sans-serif`。

西文展示和大数字可使用系统窄体优先栈：`"Avenir Next Condensed", "DIN Condensed", "Arial Narrow"`。不加载中文 webfont。

字号：
- 首页 H1：`clamp(52px, 9vw, 112px)`，窄体，最多 2 行。
- 详情标题：`clamp(34px, 5vw, 68px)`。
- 正文：15-17px，中文暗底行高 1.72。
- Meta 与标签：12-13px，必要时 uppercase 但中文不强行拉字距。

禁用斜体、负字距中文、大标题渐变文字和空洞标语。

## 4. 组件样式

- Header：透明暗场胶片栏，sticky，毛玻璃只用于导航层。
- Search：暗色输入框 + 暖黄方形搜索按钮；移动端保持一行。
- Movie card：海报为主，信息层叠在暗色面板中；hover 只做轻微上移和边框提亮。
- State button：胶囊按钮，未选暗灰，选中暖黄；必须保留计数。
- Detail page：海报 + 片名 + 评分 + 操作区在顶部形成电影档案头；下方内容模块用暗色分区，不泄露数据来源。
- Modal / Toast：暗色浮层，暖黄焦点环，保持可读。

## 5. 布局原则

- 主容器宽度 `min(1180px, calc(100% - 32px))`，不全屏。
- 首页：Header → 电影节式标题 + 搜索 → 状态栏 → 海报网格。
- 电影网格桌面 4 列，平板 3 列，手机 2 列，超窄 1 列。
- 详情页桌面 220px 海报 + 内容，移动端单列。
- 页面区块不做套卡片；只有电影、详情模块、弹窗作为卡片。

## 6. 深度层级

深度来自暗色层级、海报、内边框和少量染色阴影：
- Base：页面暗场背景。
- Surface：暗色卡片 `#171b21`。
- Elevated：`0 20px 70px rgba(0,0,0,.34)`，只给悬停卡片、弹窗、Toast。
- Inner glow：`inset 0 1px 0 rgba(255,255,255,.04)`，模拟银幕边缘反光。

## 7. Do / Don't

Do：
- 让海报成为色彩来源。
- 保持按钮、搜索、状态清楚可点。
- 保持中文正文高对比和足够行高。
- 用细网格/胶片纹理做氛围，但正文区域透明度 ≤ 3%。

Don't：
- 不做全屏首屏。
- 不用纯黑、AI 紫蓝渐变、霓虹外发光。
- 不把每块内容都包成大白卡。
- 不出现 OMDb、DeepSeek、API、模型名等技术视角文案。
- 不写“经得起时间校验”这类无增量句子。

## 8. 响应式

980px 以下：Hero 搜索堆叠、详情单列、相关电影 2 列。
620px 以下：Header 行内换行、电影网格 2 列、状态按钮可纵向排布。
430px 以下：电影网格 1 列，详情操作保持 3 列但文字更紧凑。

## 9. Motion 哲学

UI 动效服务反馈，不服务炫技：
- hover 160ms，`transform` + 边框色。
- active `scale(.98)`。
- Toast 200ms 入场。
- 支持 `prefers-reduced-motion`。
- 禁 `transition: all`。
