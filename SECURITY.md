# Security Policy

如果你发现安全问题，请不要公开提交 exploit 细节。可以通过 GitHub 私密渠道或向阳乔木公开联系方式联系维护者。

Please do not disclose exploitable details publicly. Report security issues through GitHub private channels or the maintainer contact links in the README.

## Secrets

Do not commit OMDb, DeepSeek, session, deployment, database, or analytics secrets. Use `.env.local` locally and production environment variables on servers.
